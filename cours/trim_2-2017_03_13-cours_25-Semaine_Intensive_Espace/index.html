L'espace

Objectifs

Réaliser un site sur le thème de l'espace en vous basant sur une ou plusieurs API

Attentes

  • Version en ligne
  • Code impeccable
  • Compatible navigateurs récents
  • Mise en oeuvre de ce qu'on a vu en cours
  • Fonctionnel
  • Basé sur au moins une API
  • Beau

Bonnes pratiques

  • D'autres APIs peuvent être plus intéressantes pour votre projet
  • Commencez à tester des requêtes dès le début
  • Parcourez toute les docs au moins une fois
  • Mettez en cache les résultats des APIs (tuto)
